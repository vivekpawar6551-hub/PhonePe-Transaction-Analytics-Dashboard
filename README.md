PhonePe Transaction Analytics Dashboard
Overview

This Power BI dashboard provides insights into PhonePe transactions, transaction value, user activity, payment success rates, and service-wise performance. The dashboard helps analyze transaction trends, customer behavior, and key business metrics.

Key Features
Total Transactions Analysis
Transaction Value Tracking
Unique User Analysis
Payment Status Monitoring
Service-wise Transaction Analysis
Top Users by Transaction Value
Age Segment Contribution Analysis
Weekday vs Weekend Usage Analysis
Interactive Filters and Slicers
Tools Used
Power BI
Power Query
DAX
Data Modeling
Key Insights
Weekdays contribute the majority of transactions.
Loans generate the highest transaction value.
Payment success rate remains above 95%.
Millennials and Gen X contribute the largest share of transactions.
